Variables & Data Types

10 min read
Beginner
1 exercise

What is a variable?

A variable is a named container that stores a value. Think of it like a labelled box — you give it a name, put something inside, and refer to it by that name whenever you need it.

In Python, creating a variable is as simple as writing a name, an equals sign, and a value:

variables.py
name = "Alice"
age = 25
height = 5.6
is_student = True

print(name) # Alice
print(age) # 25
Note: Python doesn't require you to declare a variable's type. It figures it out automatically from the value you assign. This is called dynamic typing.

The four basic data types

Every value in Python has a type. The four most important ones to know first are:

  • str — text, written in quotes: "hello"
  • int — whole numbers: 42
  • float — decimal numbers: 3.14
  • bool — True or False only
types.py
# Check a variable's type with type()
x = "hello"
print(type(x)) # <class 'str'>

y = 42
print(type(y)) # <class 'int'>

z = 3.14
print(type(z)) # <class 'float'>

Updating a variable

Variables can be reassigned at any time. The old value is simply replaced:

update.py
score = 0
print(score) # 0

score = 10
print(score) # 10

score = score + 5
print(score) # 15
